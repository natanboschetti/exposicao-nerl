Essa imagem foi criada a partir de uma Inteligência Artificial, é incrível como hoje podemos criar imagens como quisermos de maneira tão simples, apenas com uma frase.

O processo de criação da imagem é simples, você se conecta ao site com uma conta e insere um texto ou uma pequena frase, especificando o tipo de imagem que você deseja.
